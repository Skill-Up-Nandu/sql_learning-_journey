- **`HAVING` vs `WHERE`**:

  - `HAVING` filters **groups after aggregation**
- **Aliasing & Formatting**:
  - Renaming result columns using `AS`
  - Rounding numerical output using `ROUND(value, decimal_places)`
- **Query Structure / Order of Execution**:
  1. `SELECT`
